## Getting Started

To get a local copy and run this project follow the next steps.

### Prerequisites

npm
  ```sh
  npm install
  ```

### Setup

  ```sh
  $ git clone https://github.com/eduym17/EY-readme-template/
  ```
  
  ```sh
  cd main
  ```

### Install

  ```sh
  npm install
  ```

### Usage

  ```sh
  npm run dev
  ```

### Run tests

  ```sh
  npm run test
  ```

### Deployment

Make a pull request and when approved and merged the changes will show up on website.
